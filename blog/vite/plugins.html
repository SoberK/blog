    # plugin使用

    • 按需引入

      • 如果是分环境使用插件可以使用apply

          plugins: [
    {
      ...typescript2(),
      apply: 'build',
    },
  ],

    • 强制排序

      • 部分插件有先后顺序

         plugins: [
    {
      ...image(),
      enforce: 'pre', //在之前引用   ‘post ’在之后引用
    },
  ],

    • Vue Router

      • vite-plugin-pages (opens new window) - 以文件系统为基础的路由
      • vite-plugin-vue-layouts (opens new window) - 页面布局系统

    • Pinia (opens new window) - 直接的, 类型安全的, 使用 Composition API 的轻便灵活的 Vue 状态管理

    • unplugin-vue-components (opens new window) - 自动加载组件

    • unplugin-auto-import (opens new window) - 直接使用 Composition API 等，无需导入

    • vite-plugin-pwa (opens new window) - PWA

    • vite-plugin-vue-markdown

      - Markdown 作为组件，也可以让组件在 Markdown 中使用

      • markdown-it-prism (opens new window) - Prism (opens new window) 的语法高亮
      • prism-theme-vars (opens new window) - 利用 CSS 变量自定义 Prism.js 的主题

    • Vue I18n

      - 国际化

      • unplugin-vue-i18n (opens new window) - Vue I18n 的 Vite 插件

    • VueUse (opens new window) - 实用的 Composition API 工具合集

    • vite-ssg-sitemap (opens new window) - 站点地图生成器

    • @vueuse/head (opens new window) - 响应式地操作文档头信息

    • vite-plugin-vue-devtools (opens new window) - 旨在增强Vue开发者体验的Vite插件
